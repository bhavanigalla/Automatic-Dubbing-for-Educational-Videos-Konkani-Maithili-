\# 🎬 Automatic Dubbing for Educational Videos



\## 📌 Overview

This project automatically converts educational videos into regional languages like \*\*Konkani and Maithili\*\* using AI.



\## 🚀 Features

\- 🎙 Speech to Text using Whisper

\- 🌐 Language Translation

\- 🔊 Text to Speech

\- 🎬 Video Dubbing

\- 💻 Streamlit Web Application



\## 🛠 Tech Stack

\- Python

\- Streamlit

\- OpenAI Whisper

\- MoviePy

\- gTTS

\- Deep Translator



\## ▶️ How to Run



```bash

pip install -r requirements.txt

streamlit run app.py
